# Warehouse Management System

This repository contains a sample implementation of a warehouse workforce management system. The project is based on the requirements described in [docs/SPECIFICATION.md](docs/SPECIFICATION.md).

## Getting Started

1. Install dependencies (requires internet access):
   ```
   npm install
   ```
2. Build and run in development mode:
   ```
   npm run dev
   ```

The server exposes a simple health endpoint at `/`.
